Método del Punto fijo

El método del punto fijo es una técnica iterativa utilizada para encontrar soluciones de ecuaciones no lineales. Se basa en transformar la ecuación original en una forma equivalente g(x) = x, donde g(x) es una función conocida como la función de iteración. En cada iteración, se toma un valor inicial X0 y se actualiza sucesivamente mediante Xn+1 = g(Xn) hasta que la secuencia converge a un punto fijo de g(x). El método es ampliamente utilizado en análisis numérico y tiene aplicaciones en la resolución de ecuaciones no lineales en diversas disciplinas científicas e ingenieriles.

Método de Newton-Raphson

El método de Newton-Raphson es una técnica iterativa utilizada para encontrar soluciones de ecuaciones no lineales. Se basa en la idea de aproximar la función por su tangente en el punto actual, y luego encontrar la intersección con el eje x de la tangente para obtener el siguiente punto de iteración.

Método de Bisección

El método de bisección es un algoritmo que encuentra las raíces de una ecuación no lineal al dividir repetidamente un intervalo y seleccionar el subintervalo en el que cambia de signo.

integración numérica

La integración numérica es un método para calcular el valor numérico de una integral definida. La integración numérica es ampliamente utilizada en el análisis numérico y en el cálculo de áreas y volúmenes de regiones e hipervolúmenes, aqupi la usamos para calcular el área bajo la curva.

Derivación numérica

La derivación numérica es un método de diferenciación numérica para encontrar la pendiente de una curva en un punto dado. La diferenciación numérica es una técnica que se utiliza para aproximar la derivada de una función en un punto específico mediante diferencias finitas.

Método de Gauss-Seidel

El método de Gauss-Seidel es un algoritmo iterativo utilizado para resolver sistemas de ecuaciones lineales. Este método pertenece a la categoría de métodos iterativos para la solución de sistemas lineales y se utiliza para encontrar soluciones aproximadas en lugar de soluciones exactas.

Método de Gauss-Jacobi

El método de Gauss-Jacobi es un algoritmo iterativo utilizado para resolver sistemas de ecuaciones lineales. Al igual que Gauss-Seidel, este método pertenece a la categoría de métodos iterativos para la solución de sistemas lineales y se utiliza para encontrar soluciones aproximadas en lugar de soluciones exactas.

Polinomio de Lagrange con eliminación de Gauss-Jordan

El Polinomio de Lagrange con eliminación de Gauss-Jordan se utiliza para encontrar un polinomio interpolante que pase exactamente por un conjunto dado de puntos. Este método combina el polinomio de Lagrange y el proceso de eliminación de Gauss-Jordan para encontrar los coeficientes del polinomio.

Polinomio de Lagrange por método directo

El Polinomio de Lagrange por método directo se utiliza para encontrar un polinomio interpolante que pase exactamente por un conjunto dado de puntos. Este método utiliza la fórmula de Lagrange para encontrar los coeficientes del polinomio.
